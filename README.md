# AI Quiz Platform

An AI-powered quiz and interview preparation platform.

## Tech Stack

**Backend:** Django · DRF · PostgreSQL · JWT · Google Gemini · LangChain · FAISS  
**Frontend:** React · Vite · Tailwind CSS · React Query · Axios

## Features

- 📄 Upload PDFs and extract text automatically
- 🤖 Generate MCQ quizzes with Groq AI (RAG pipeline)
- ⏱️ Timed quiz interface with question navigation
- 📊 Analytics dashboard with charts and leaderboard
- 🔐 JWT authentication with auto token refresh
